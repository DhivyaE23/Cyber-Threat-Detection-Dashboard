# Cyber Threat Detection Dashboard
## Overview
A cybersecurity dashboard that analyzes network logs, detects suspicious activities, generates alerts, and visualizes attack patterns.

## Features
- Failed login detection
- High traffic anomaly detection
- Alert generation
- Protocol distribution visualization
- Interactive dashboard

## Technologies Used
- Python
- Pandas
- Streamlit
- Plotly

## Project Structure

Cyber-Threat-Detection-Dashboard/
│
├── data/
│   └── network_logs.csv
├── app.py
├── threat_detection.py
├── test.py
├── requirements.txt
└── README.md

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Future Enhancements

- Real-time log monitoring
- Machine Learning anomaly detection
- Email alerts
